=head1 NAME

xdg-startup - an XDG compliant X startup notification launcher

=head1 SYNOPSIS

B<xdg-startup> [I<OPTIONS>] [I<APPID>|I<COMMAND>]

=head1 DESCRIPTION

B<xdg-startup> is a script that simply executes L<xdg-launch(1)> with
the appropriate arguments (B<--startup>).
See L<xdg-launch(1)/COMMAND OPTIONS> for more information.

=head1 FILE FORMAT

For the format of auto-start files, see L<xdg-autostart(1)>.

=head1 BUGS

B<xdg-startup> has no known bugs.
Please report bugs to the source code repository at
L<https://github.com/bbidulock/xdg-launch/issues>.

=head1 SEE ALSO

L<xdg-launch(1)>.
