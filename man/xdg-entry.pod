=head1 NAME

xdg-entry - an XDG compliant desktop entry inspection tool

=head1 SYNOPSIS

B<xdg-entry> [I<OPTIONS>] [I<APPID> [I<FILE>|I<URL>]]

B<xdg-entry> [I<OPTIONS>] -c I<command and arguments>

=head1 DESCRIPTION

B<xdg-entry> is a script that simply executes L<xdg-launch(1)>
with the appropriate arguments (B<--info>).

=head1 BUGS

B<xdg-entry> has no known bugs.
Please report bugs to the source code repository at
L<https://github.com/bbidulock/xdg-launch/issues>.

=head1 SEE ALSO

L<xdg-launch(1)>.
